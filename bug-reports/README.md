# Bug Reports

자동차보험 업무 흐름에서 발생할 수 있는 오류 사례를 QA 관점으로 정리한 버그리포트 문서입니다.

본 문서는 특정 보험사의 내부 시스템을 설명하기 위한 것이 아니라, 보험 CS/운영 경험에서 확인한 오류 사례를 요구사항 기반 QA 문서 형식으로 재구성한 포트폴리오입니다.

## 작성 기준

* 특정 보험사명, 내부 시스템명, 벤더명은 사용하지 않습니다.
* 기능 개선 제안이 아니라 요구사항과 실제 동작의 차이를 기준으로 작성합니다.
* 재현 절차, 기대 결과, 실제 결과, 고객 영향을 명확히 구분합니다.
* 고객센터 문의, 가입 중단, 결제 혼란 등 실제 고객 영향 관점도 함께 정리합니다.

## 문서 목록

BUG-001_갱신가입_차량부속정보_중복오류.md  
- 갱신가입 시 외부 차량정보 조회 결과와 기존 입력 부속정보가 중복 반영되어 다음 단계로 진행되지 않는 사례

BUG-002_당일개시계약_단기운전자특약_가입가능오류.md  
- 당일 개시 계약에서 실시간 단기운전자 특약 가입 불가 조건임에도 모바일에서 결제까지 진행되는 사례

## Bug Report 항목

각 문서는 아래 항목을 기준으로 작성합니다.

* Summary
* Environment
* Precondition
* Steps to Reproduce
* Expected Result
* Actual Result
* Severity
* Priority
* Customer Impact
* QA Note
